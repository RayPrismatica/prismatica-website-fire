# CMO Advisor Reference Materials

This directory contains strategic marketing documentation and reference materials for Prismatica Labs.

## Contents

### Core Documents

**`messaging-frameworks.md`** - Key messages, positioning statements, value propositions, and elevator pitches for different audiences.

**`competitive-landscape.md`** - Market analysis, competitor positioning, differentiation strategies, and market opportunities.

**`content-templates.md`** - Templates for blog posts, case studies, whitepapers, email campaigns, and social media posts.

## How to Use

These references support the CMO advisor and sub-skills by providing:

1. **Messaging Frameworks** - Ensure consistent value propositions across all channels
2. **Competitive Intelligence** - Inform differentiation and positioning decisions
3. **Content Templates** - Accelerate content creation while maintaining quality

## Updating References

References should be updated:
- **Quarterly**: Competitive landscape, market trends
- **Bi-annually**: Messaging frameworks, positioning
- **As needed**: Content templates, new campaign types

## Integration with Sub-Skills

- **brand-voice-enforcer**: Uses messaging frameworks to validate voice consistency
- **content-strategist**: Uses templates and frameworks for content creation
- **market-researcher**: Maintains competitive landscape and market intelligence
- **campaign-planner**: References messaging for campaign development
- **seo-optimizer**: Uses keyword research integrated with messaging
