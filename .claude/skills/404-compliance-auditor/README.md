# 404 Compliance Auditor

**Type:** Project Skill
**For:** CMO Advisor Agent
**Purpose:** Audit and maintain 100% accuracy between website architecture and the 404 error page navigation tree

## When to Use

- After adding/removing pages from the website
- After modifying modal structure in `/about`
- After adding/removing bento boxes in `/solutions`
- Quarterly maintenance audits
- When 404 page feels outdated

## What It Does

1. **Scans entire website architecture** to build current sitemap
2. **Compares against 404 page** to identify gaps
3. **Verifies tone compliance** (Rory Sutherland style)
4. **Generates prioritized task list** with specific fixes

## Output

- Compliance score (0-100)
- Categorized issues (Critical â†’ Low priority)
- Specific line numbers for fixes
- Estimated time to repair
- Before/after examples for tone violations

## Example Usage

```
Use the 404-compliance-auditor skill to verify the 404 page accurately reflects the current website architecture
```

## Success Criteria

**100/100 means:**
- Zero broken links
- All routes covered
- All descriptions follow Rory guidelines
- Counts are accurate
- Tone is consistently anti-marketing
